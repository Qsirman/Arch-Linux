# CMDX

### Here I collect some software to display some x-files like pdf,picture and etc. in a terminal.

## Picture(Eog)
### Install
```
sudo pacman -S eog
```
### Usage
```
eog $(PATH OF YOUR FILE)
```

## PDF(Okular)
### Install
```
sudo pacman -S okular
```
### Usage
```
okular $(PATH OF YOUR FILE)
```
